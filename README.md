# 电商流量数据分析与优化建议生成系统

## 项目概述

这是一个基于Spring Boot + Vue.js的电商流量数据分析系统，能够处理Excel文件中的流量数据，进行异常检测，抓取文章内容，并生成优化建议。

## 功能特性

### 🔍 核心功能
- **Excel数据解析**: 支持上传包含流量数据的Excel文件
- **流量异常检测**: 使用统计学方法(Z-score、IQR)检测异常流量
- **内容抓取**: 自动抓取文章链接的内容
- **智能分析**: 基于内容和流量数据的关联分析
- **优化建议**: 为每篇文章生成个性化优化建议

### 📊 数据分析
- 支持7天和14天的阅读量、互动量、分享量分析
- 异常状态分类：正常、异常好、异常差
- 实时统计和可视化图表展示

### 💡 优化建议类型
- **标题优化**: 基于关键词和长度分析
- **内容优化**: 互动性和吸引力提升建议
- **发布策略**: 时间和平台优化建议

## 技术栈

### 后端
- **Spring Boot 4.0.1**: 主框架
- **Spring Data JPA**: 数据持久化
- **H2 Database**: 内存数据库(开发环境)
- **Apache POI**: Excel文件处理
- **JSoup**: 网页内容抓取
- **Apache Commons Math**: 统计分析

### 前端
- **Vue 3**: 前端框架
- **TypeScript**: 类型安全
- **Element Plus**: UI组件库
- **ECharts**: 数据可视化
- **Axios**: HTTP客户端
- **Vite**: 构建工具

## 快速开始

### 环境要求
- Java 17+
- Node.js 16+
- Maven 3.6+

### 1. 启动后端服务
```bash
# 方式1: 使用批处理文件
start-backend.bat

# 方式2: 手动启动
cd BackEnd
./mvnw spring-boot:run
```

### 2. 启动前端服务
```bash
# 方式1: 使用批处理文件
start-frontend.bat

# 方式2: 手动启动
cd FrontEnd
npm install
npm run dev
```

### 3. 访问系统
- 前端地址: http://localhost:3000
- 后端API: http://localhost:8080
- H2数据库控制台: http://localhost:8080/h2-console

## 使用流程

### 1. 数据上传
1. 访问系统首页
2. 点击上传区域或拖拽Excel文件
3. 点击"开始分析"按钮

### 2. 查看分析结果
1. 进入"数据分析"页面查看整体统计
2. 查看异常状态分布图表
3. 浏览文章列表和详细信息

### 3. 异常检测结果
1. 进入"异常检测"页面
2. 筛选查看异常好/异常差的文章
3. 查看每篇文章的优化建议

## 异常检测算法

### 1. Z-Score检测
- 计算每个指标的Z分数
- 阈值: |Z| > 2.0 视为异常

### 2. IQR检测
- 使用四分位距检测离群值
- 异常范围: Q1-1.5×IQR 或 Q3+1.5×IQR

### 3. 综合评分
- 结合多个指标进行综合判断
- 权重分配: 分享量 > 互动量 > 阅读量

## 优化建议生成

### 异常好文章
- 分析成功要素
- 提供复制策略
- 建议扩大影响力

### 异常差文章
- 标题优化建议
- 内容改进方案
- 发布策略调整

### 正常文章
- 提升空间分析
- 常规优化建议

## 注意事项

1. **文件大小限制**: Excel文件不超过50MB
2. **网络抓取**: 部分网站可能有反爬虫机制
3. **数据安全**: 使用内存数据库，重启后数据会丢失
4. **并发处理**: 内容抓取采用异步处理，提高效率
