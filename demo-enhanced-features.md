# 增强功能演示指南

## 🎯 演示目标

本指南将演示系统的新增功能，包括AI智能建议、图片分析、增强异常检测等特性。

## 📋 演示准备

### 1. 启动系统
```bash
# 使用一键启动脚本
start-enhanced-system.bat

# 或手动启动
cd BackEnd && ./mvnw spring-boot:run
cd FrontEnd && npm run dev
```

### 2. 访问地址
- 前端: http://localhost:3000
- 后端API: http://localhost:8080/api
- 增强API: http://localhost:8080/api/enhanced

## 🔍 功能演示步骤

### 第一步：数据上传与基础分析

1. **上传Excel文件**
   - 访问首页，上传包含流量数据的Excel文件
   - 观察系统自动解析和验证过程
   - 查看上传成功提示

2. **查看基础统计**
   - 进入"数据分析"页面
   - 观察总体统计数据
   - 查看异常状态分布图表

### 第二步：增强异常检测演示

1. **多算法检测结果**
   - 进入"异常检测"页面
   - 查看异常好/异常差文章列表
   - 点击文章查看详情

2. **详细异常分析**
   - 在详情弹窗中切换到"原因分析"标签
   - 观察多维度异常检测结果：
     - Z-score分析
     - IQR检测
     - Isolation Forest评分
     - 百分位排名

### 第三步：图片分析功能演示

1. **图片自动下载**
   - 查看文章详情中的图片分析部分
   - 观察系统自动下载的图片信息
   - 查看图片类型分类结果

2. **图片质量评估**
   - 查看图片数量统计
   - 观察图片类型分布（商品图、场景图、细节图等）
   - 查看下载状态和本地存储路径

### 第四步：AI智能建议演示

1. **查看AI建议**
   - 在文章详情中切换到"AI智能建议"标签
   - 观察AI生成的六大分析模块：
     - 智能内容分析
     - 竞品对标分析
     - 用户行为预测
     - 个性化优化路径
     - 风险评估预警
     - A/B测试建议

2. **AI建议特色功能**
   - 查看标题智能评分（吸引力、情感强度等）
   - 观察用户行为预测结果
   - 查看个性化优化路径的具体步骤

### 第五步：分离式建议展示

1. **三种建议类型对比**
   - **原因分析**: 数据驱动的异常原因分析
   - **优化建议**: 传统的基于规则的建议
   - **AI智能建议**: 机器学习生成的个性化建议

2. **建议内容差异**
   - 对比三种建议的内容深度和个性化程度
   - 观察AI建议的创新性和实用性

## 🧪 API功能测试

### 1. 获取AI建议
```bash
# 获取文章AI建议
curl -X GET "http://localhost:8080/api/enhanced/articles/1/ai-suggestions"

# 重新生成AI建议
curl -X POST "http://localhost:8080/api/enhanced/articles/1/regenerate-ai"
```

### 2. 图片分析API
```bash
# 获取图片分析结果
curl -X GET "http://localhost:8080/api/enhanced/articles/1/images-analysis"
```

### 3. 批量处理
```bash
# 批量生成AI建议
curl -X POST "http://localhost:8080/api/enhanced/batch-generate-ai?limit=5"

# 获取增强统计信息
curl -X GET "http://localhost:8080/api/enhanced/statistics"
```
